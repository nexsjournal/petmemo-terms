PetMemo 隐私政策

生效日期：2026 年 6 月 12 日
最后更新：2026 年 6 月 12 日

引言

欢迎使用 PetMemo（以下简称「本应用」或「我们」）。

我们重视并保护用户个人信息与隐私安全。本隐私政策用于说明您在下载、安装、访问或使用 PetMemo 过程中，我们如何收集、使用、存储、共享和保护您的信息，以及您享有的相关权利。请您在使用本应用前仔细阅读并充分理解本政策。您开始使用本应用，即视为您已阅读并同意本政策。

PetMemo 是一款面向宠物主人的本地记录工具，帮助您记录宠物档案、成长瞬间、散步轨迹与足迹，并生成可分享的回忆卡片。我们遵循 Local-first（本地优先） 原则：您的宠物隐私数据默认保存在您的设备本地，我们不会将其上传至开发者自建业务服务器进行持久化存储。

1. 我们处理的信息类型

根据当前版本功能，我们主要处理以下信息：

1.1 宠物与记录信息（本地）

您主动创建或录入的内容，保存在设备本地（SwiftData），包括但不限于：

  • 宠物档案：名称、物种（猫/狗）、品种、性别、生日、体重、简介、头像等
  • 成长记录：记录类型、标题、正文、日期、体重、标签、关联照片等
  • 散步记录：开始/结束时间、轨迹坐标点、距离、时长、备注、回忆文案等
  • 足迹地点：地点名称、经纬度、访问次数、备注等

1.2 相册媒体信息

当您使用 PhotosPicker 选择宠物头像或成长记录照片时，本应用会读取您选中的照片内容，并将其复制保存至本应用沙盒目录（Documents/Media），用于在应用内展示。

  • 我们仅处理您主动选择的照片，不会在后台扫描整个相册。
  • 当前版本不会将照片上传至开发者自建服务器。

1.3 位置信息

当您使用地图遛弯、轨迹录制或足迹打卡功能时，在您授权「使用期间定位」后，我们会收集：

  • 设备实时位置坐标（用于记录散步轨迹、计算距离与时长）
  • 通过 Apple 系统地理编码服务将坐标转换为地点名称（如公园、街道名）

我们不会申请「始终」定位权限；定位仅在您使用相关功能且授权时获取。

1.4 回忆文案生成相关数据

为生成散步或成长记录后的回忆短文案，本应用可能通过以下方式处理有限信息：

方式 传输内容 说明
本地模板 不联网 默认路径，在设备本地根据模板生成文案
服务端 API（可选） 宠物名称、物种、记录类型、距离/时长、地点名称、时段等文本与数值 不传输照片、完整轨迹坐标或健康诊断信息

服务端当前主要用于文案生成代理；不会持久化存储您的宠物隐私档案。

1.5 应用运行与偏好信息（本地）

保存在设备本地（UserDefaults 等），包括：

  • 应用设置：通知开关、定位开关、距离单位、外观模式等
  • 用户资料（可选）：您在设置中填写的昵称、邮箱（仅本地存储，非账号体系）
  • 订阅权益状态标记（用于 Pro 权益判断，以 Apple StoreKit 交易校验为准）
  • 首次启动种子数据标记等运行状态

1.6 我们不主动收集的信息

在当前版本中：

  • 不提供账号注册、登录体系，不主动收集手机号、密码
  • 不接入 HealthKit，不提供医疗诊断
  • 不上传宠物照片、完整轨迹、成长记录正文至开发者自建数据库
  • 不收集通讯录、日历、麦克风、相机（拍照）等与核心功能无关的信息

您在设置中自愿填写的邮箱仅保存在本地，用于您的个人备忘，我们不会将其用于营销或共享给第三方。

2. 我们如何使用信息

我们仅在实现本应用功能所必需范围内处理您的信息，主要用于：

  • 展示与管理宠物档案、成长时间线、散步记录与足迹地图
  • 录制散步轨迹、统计距离与时长、聚合常去地点
  • 生成回忆卡片文案（本地模板或服务端 API）
  • 渲染并支持您分享回忆卡片、导出 JSON 数据
  • 提供 Pro 订阅购买、权益校验与恢复购买
  • 在您授权后发送本地/远程通知提醒（如散步提醒）
  • 保障应用运行安全、性能优化与故障排查

我们不会将您的信息用于与上述目的无关的用途。若未来用途发生实质变化，我们将依法另行告知并征得必要同意。

3. 权限使用说明

3.1 定位权限（使用期间）

用于地图遛弯、轨迹录制、足迹地点聚合与打卡。若您拒绝授权，相关功能将无法正常使用，但不影响宠物档案与成长记录等不依赖定位的功能。

3.2 相册权限（读取）

用于您主动选择宠物头像与成长记录照片。若您拒绝授权，将无法从相册选取图片，但仍可仅使用文字记录。

说明：本应用通过系统 PhotosPicker 访问相册，遵循 iOS 系统的有限访问或完整访问规则，由您在系统弹窗中选择授权范围。

3.3 通知权限

用于向您发送提醒类通知（如散步提醒、成长记录提醒）。该权限为可选，拒绝授权不影响核心记录功能。

3.4 其他权限

根据当前版本实现，我们未主动申请相机、麦克风、通讯录、日历、蓝牙、运动与健身等与核心功能无关的系统权限。

4. 网络访问与对外传输

4.1 Apple 系统服务

本应用在以下场景可能发生网络访问：

  • 应用内购买与订阅：通过 Apple StoreKit 与 App Store 进行商品查询、购买、恢复购买及交易校验（月会员、年会员、终身会员）。
  • 地图与地理编码：通过 Apple MapKit / CoreLocation 地理编码服务，将坐标转换为地点名称。
  • iCloud 照片（系统能力）：若您从 iCloud 相册选取照片，系统可能在您操作时下载相应资源。

4.2 回忆文案服务（可选）

当您使用服务端生成回忆文案时，本应用会向配置的 API 地址发送有限 JSON 数据（如宠物名称、物种、散步距离等），并接收生成的短文案。不会传输照片或完整轨迹数据。

若网络不可用或您未配置远程服务，本应用将自动使用本地模板生成文案，不影响记录保存。

4.3 分享与导出

当您主动使用系统分享面板（ShareLink）分享回忆卡片图片或导出 JSON 文件时，相关数据将由您选择的第三方应用（如微信、邮件、文件）接收并处理，该处理受第三方平台规则约束。

4.4 法律文档与外链

当您点击应用内「隐私政策」「用户协议」等链接时，会访问对应网页地址。

4.5 关于自建服务器与数据上传

依据当前版本实现：

  • 宠物档案、成长记录、散步轨迹、照片等媒体默认仅保存在您的设备本地。
  • 服务端（如已部署的 AI 文案网关）不持久化您的宠物隐私档案；仅处理生成文案所需的最小字段。
  • 我们未发现将相册媒体或完整本地数据库上传至开发者自建业务服务器的功能通道。

5. 信息存储与保存期限

5.1 存储位置

您的数据主要保存在本地设备，包括但不限于：

  • SwiftData：宠物档案、成长记录、散步会话、足迹地点等结构化数据
  • 应用沙盒（Documents/Media）：宠物头像、成长记录照片
  • UserDefaults：应用偏好设置、可选昵称/邮箱
  • 临时目录：回忆卡片渲染图片、数据导出 JSON 临时文件

5.2 保存期限

  • 本地数据在实现对应功能所需期限内保存，直至您主动删除记录、删除宠物档案或卸载应用。
  • 卸载应用后，应用沙盒内数据通常会被 iOS 系统移除。
  • 您可通过应用内「导出数据」功能将 JSON 备份至您选择的位置；也可通过系统分享保存回忆卡片。

5.3 删除

您可以在应用内删除单条成长记录、散步记录、宠物档案；删除宠物将级联删除其关联记录。相关本地媒体文件将一并清理。

6. 信息共享与披露

6.1 我们不会出售个人信息

我们不会出售、出租您的个人信息，也不会将您的个人信息用于广告交易。

6.2 必要共享情形

在以下必要场景，可能发生信息共享或披露：

  • 与 Apple 提供的系统服务交互（StoreKit、MapKit、CoreLocation、PhotosUI 等）
  • 与文案生成服务交互时，传输第 1.4 节所述有限字段（不含照片与完整轨迹）
  • 您主动触发分享至第三方应用
  • 法律法规或有权机关依法要求

7. 数据安全

我们采取合理可行的安全措施保护您的信息，包括但不限于：

  • 遵循 iOS 沙盒与系统安全机制
  • 最小必要权限原则
  • 本地数据存储，默认不上云
  • 持续修复已知安全与稳定性问题

请您理解，任何技术措施都无法保证绝对安全。您应妥善保管设备并启用系统安全功能（如锁屏密码、Face ID / Touch ID）。

8. 您的权利

在适用法律允许范围内，您可行使以下权利：

  • 查询与管理：通过应用内界面查看、编辑宠物档案与各类记录
  • 删除：删除指定记录、宠物档案，或卸载应用
  • 导出：通过「设置 → 导出数据」获取 JSON 格式备份
  • 撤回权限：在 iOS「设置 → 隐私与安全性」中关闭定位、相册、通知等权限
  • 恢复购买：在 Pro 订阅页面使用「恢复购买」功能

如您希望行使其他隐私权利（如投诉、建议），可通过本政策「联系我们」方式反馈。

9. 未成年人保护

若您为未成年人，请在监护人阅读并同意本政策后使用本应用。监护人应对未成年人使用行为进行指导与监督。

12. 其他重要信息

12.1 数据最小化原则

我们遵循「数据最小化」原则，只收集和处理提供服务所必需的最少信息。宠物隐私数据以本地存储为默认策略。

12.2 健康信息声明

本应用中的体重、疫苗、健康类记录仅供您个人备忘，不构成兽医诊断、治疗或医疗建议。如有健康问题，请咨询专业兽医。

12.3 政策更新

我们可能适时更新本政策。更新后将在应用内或网页公布，重大变更将以合理方式提示。若您在更新生效后继续使用本应用，视为接受更新后的政策。
